Cara instalasinya

1. Buka XAMPP Server Versi (bebas)

2. Buka "TEKS EDITOR" Notepad++ ATAU SUBLIME TEKS 3 / DLL. saya saranin buat pakai VSCode

3. Clone repository ini  https://github.com/4lifbima/osrs.git

4. Download file zip/unduh winrar

5. Ekstrak file dan salin folder "osrs".

6. Paste di dalam direktori root/tempat Anda menginstal xammp disk lokal C: drive D: drive E: paste: (untuk xampp/htdocs, 

7. Buka PHPMyAdmin (http://localhost/phpmyadmin)

8. Buat database dengan nama osrs_db

6. Impor file osrs_db.sql (diberikan di dalam paket zip di folder file SQL)

7.Jalankan skrip http://osrs
